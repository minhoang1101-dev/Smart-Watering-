# Smart Watering System

ESP32 smart watering project with MicroPython and Arduino examples.

## Folders

- `micropython/main.py`: main MicroPython program for ESP32.
- `micropython/connect_all_sensors.py`: full sensor connection logic.
- `micropython/tests/`: separate tests for each sensor/component.
- `diagnostics/`: no-sensor diagnostic scripts that can run on a computer.
- `smart_watering_esp32/`: Arduino `.ino` version.

## Components

- ESP32 board
- Soil moisture sensor
- Relay module
- Mini water pump
- DHT11 or DHT22 sensor
- LED with 220 ohm resistor
- Jumper wires, breadboard, external pump power supply, USB cable
